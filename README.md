#  História do HTML

Este projeto foi desenvolvido com o objetivo de apresentar a história do HTML, suas principais versões e sua importância na criação de páginas web.

---

##  Sobre o projeto

O site foi criado utilizando apenas HTML e tem como finalidade explicar de forma simples e direta:

- O que é HTML  
- Quem criou o HTML  
- Como ele funciona  
- A evolução das versões ao longo do tempo  

---

##  Conteúdo abordado

- HTML (HyperText Markup Language)
- Criação por Tim Berners-Lee em 1991
- Estrutura básica de páginas web
- Principais versões:
  - HTML 1.0
  - HTML 2.0
  - HTML 3.2
  - HTML 4.01
  - XHTML
  - HTML5
