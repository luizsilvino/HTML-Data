História do HTML

O que é HTML?

HTML (HyperText Markup Language) é uma linguagem de marcação utilizada para criar páginas web. Ela permite estruturar conteúdos como textos, imagens, vídeos e links.

O HTML não é uma linguagem de programação, pois não possui lógica como variáveis ou condições. Ele serve para organizar o conteúdo que será exibido no navegador.

Quem criou o HTML?

O HTML foi criado por Tim Berners-Lee em 1991, enquanto trabalhava no CERN. Seu objetivo era facilitar o compartilhamento de informações na internet.

Como o HTML funciona?

O HTML utiliza "tags" (etiquetas) para estruturar o conteúdo. Cada tag tem uma função específica.

Exemplos de tags:

  • <h1> → título
  • <p> → parágrafo
  • <a> → link
  • <img> → imagem

Principais versões do HTML

  • HTML 1.0 (1991) – versão inicial, muito simples
  • HTML 2.0 (1995) – primeira versão oficial padronizada
  • HTML 3.2 (1997) – suporte a tabelas e imagens
  • HTML 4.01 (1999) – separação de conteúdo e design (CSS)
  • XHTML (2000) – versão mais rígida e organizada
  • HTML5 (2014) – versão moderna com suporte a vídeos, áudios e jogos

Importância do HTML

O HTML é essencial para a criação de qualquer site na internet. Ele funciona junto com o CSS (responsável pelo design) e o JavaScript (responsável pela interatividade).
